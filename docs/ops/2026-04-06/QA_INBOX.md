- 기능
    - AI 시장 요약 페이지
        - 현재는 최신 해석이 완료된 뉴스를 바탕으로 정리한 카드만 보임
        - 카드를 좌우로 넘길 수 있도록 하고 기능을 추가해야 함
            - 좌로 넘기면
                - 어제의 예측 성공 여부 및 어제의 시황 분석 카드
            - 우로 넘기면
                - 내일의 시장을 예측하는 카드 (확률로)
                - 선물 가격과 최신뉴스를 종합해 분석할 것
                - 분석에 관한 알고리즘 및 수치는 Crisis Opportunity(Customized GPT)에게 정하도록 하는 prompt 마련할 것
            - 가운데 페이지
                - 보다 현실을 반영하고 앞으로의 1~2시간, 즉 가까운 미래를 예측하는 카드
                - 가독성, 임팩트가 '빡!' 올 수 있게 유머있는 카드가 돼야함
        - 상세 페이지 진입 장벽을 낮춰야함
            - 현재는 login 해야 들어갈 수 있음
            - 일단은 모두 접근 가능하도록 수정
                - 추후 naver, kakao, google 자동 로그인 기능 도입해서 회원만 볼 수 있도록 변환
    - 메인페이지 하단 뉴스 테이블
        - 당일 생성된 뉴스만 보관할 것
        - 전일 생성된 뉴스는 자동 삭제
    - 뉴스 수집시 키워드 선정 다변화, 다각화
        - 특히 정치, 지정학적 이슈와 관련된 키워드로 시장민감도가 높은 순으로 뉴스 수집 필요함

        - 일별 요약, 월별 요약의 비용이 다른 점 개선
            - 과거의 기록 다 지우고 2026년 4월부터 기록할 것
            - 예상 비용이 실제 Open AI 사이트의 Billing 내역과 너무 다름
    - 시장 예측에 필요한 지표 다변화, 다각화
